# personal/ — "myself" (private)

Your own knowledge base — the "extra branch for my private projects." This folder is
**gitignored**: nothing in it is committed or sent anywhere. App spec:
[[../../_apps/personal]].

Suggested shape (create as you go — Claude will file into these via `/brain-review`):

```
personal/
├── education/     courses, books, what you're learning + notes
├── goals/         goals you set and revise (linked decisions when they change)
├── journal/       daily notes (YYYY-MM-DD)
├── health/        habits, health decisions
└── decisions/     life decisions worth remembering (LRN-/HLT-/MNY-/REL- ids)
```

Capture with `/brain-capture`, reflect weekly with `/brain-review`, log real life
decisions with `/brain-decision`. Domains for personal decisions extend the standard:
`learning | health | money | relationships`.
