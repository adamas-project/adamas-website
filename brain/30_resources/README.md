# 30_resources/ — reference material by topic

**Resources** are reference you'll reuse across projects — not tied to one goal.
Organize by topic, one folder or note per topic. Examples:

- `local-first-ai/` — notes, links, the cost case (see `/guides/local-first-ai-for-founders.html`)
- `decision-ledger-standard/` — the method itself (`/standard.html`, schema)
- `obsidian/` — vault tips, plugins you use
- `prompts/` — reusable prompts/snippets for Claude

Pulled-in reference (Drive docs, saved articles) lands here after a `/brain-review`.
Anything that becomes project-specific moves into that project; anything stale moves
to `40_archive/`.
